Zoom

A simple visualisation of random polka dots with dragging and zooming interaction based on a bl.ock by @mbostock (GPL-3.0).

Features

drag, zoom

  • d3-array — d3.range — Array transformations
  • d3-drag — d3.drag — Drag and drop
  • d3-selection — d3.select and d3.event — Select elements
  • d3-zoom — d3.zoom — Pan and zoom
