Sort

A bar chart visualising the frequency of letters in the English language with sorting capabilities based on a bl.ock by @mbostock (GPL-3.0).

Data

Comma-separated values (CSV) with 26 rows and two columns:

  • letter — Each letter in the alphabet
  • frequency — Number describing how frequent the corresponding letter occurs in the English language

Features

  • d3-tip — Tooltips!
  • d3-array — d3.max and d3.ascending — Array statistics and searching
  • d3-axis — d3.axisBottom and d3.axisLeft — Reference marks for scales
  • d3-request — d3.csv — Loading files
  • d3-scale — d3.scaleBand and d3.scaleLinear — Position encodings
  • d3-selection — d3.select — Select elements
  • d3-timer — d3.timeout — Efficient animation queueing
