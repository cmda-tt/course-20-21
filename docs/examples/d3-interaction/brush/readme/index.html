Brush

An area chart with a brush visualising monthly stock data of S&P 500 based on a bl.ock by @mbostock (GPL-3.0).

Data

Comma-separated values (CSV) with 123 rows and two columns:

  • date — Month and year
  • price — Stock price as a number in dollars

Features

  • d3-array — d3.max and d3.extent — Array statistics
  • d3-axis — d3.axisBottom and d3.axisLeft — Reference marks for scales
  • d3-brush — d3.brushX — One- or two-dimensional region selections
  • d3-request — d3.csv — Loading files
  • d3-scale — d3.scaleTime and d3.scaleLinear — Position encodings
  • d3-selection — d3.select and d3.event — Select elements
  • d3-shape — d3.area and d3.curveMonotoneX — Graphical primitives
  • d3-time-format — d3.timeParse — Parse and format times
  • d3-zoom — d3.zoom and d3.zoomIdentity — Pan and zoom
