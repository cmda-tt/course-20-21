Representation

An grouped or stacked bar chart visualising random data based on a bl.ock by [@mbostock][block-author] (GPL-3.0).

Features

  • d3-axis — d3.axisBottom — Reference marks for scales
  • d3-array — d3.max, d3.range, and d3.transpose — Array statistics and transformations
  • d3-scale — d3.scaleBand, d3.scaleLinear, and d3.scaleOrdinal — Position encoding
  • d3-selection — d3.select and d3.selectAll — Select elements
  • d3-shape — d3.stack — Graphical primitives
  • d3-timer — d3.timeout — Efficient animation queueing
